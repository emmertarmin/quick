# @emmertarmin/quick

CLI for deploying and managing self-hosted Quick static sites.

## Install

```sh
bun add -g @emmertarmin/quick
```

## Usage

```sh
quick --help
quick config set remote https://quick.example.com
quick init
quick deploy .
```

## Requirements

- Bun 1.0+
